• What is the definition of server port?

A server port is a virtual endpoint in a network connection, used by a computer or server to communicate with other devices.

• What is server port range?

The range of server ports is from 0 to 65535.

• How many bits does number above represent?

The server port numbers are represented by a 16bit number.

• State 10 common server ports

Port 20/21: FTP (File Transfer Protocol)
Port 22: SSH (Secure Shell)
Port 25: SMTP (Simple Mail Transfer Protocol)
Port 53: DNS (Domain Name System)
Port 80: HTTP (HyperText Transfer Protocol)
Port 110: POP3 (Post Office Protocol v3)
Port 143: IMAP (Internet Message Access Protocol)
Port 443: HTTPS (HTTP Secure)
Port 3306: MySQL
Port 3389: RDP (Remote Desktop Protocol)
